# Ralph Claude

An Electron desktop app that orchestrates fresh Claude Code instances per spec-kit phase — combining the Ralph Wiggum approach (clean context per unit of work) with GitHub spec-kit's structured specification system. Full agent step visibility including subagents, tool calls, and real-time streaming.
